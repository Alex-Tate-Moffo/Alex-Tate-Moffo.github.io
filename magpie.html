Magpie: Media Collection App

Devloped by Alex Tate-Moffo, Jaid Basket, and Thomas Marshall, Magpie: Media Collection App was a UI design project showcasing an original app idea. The demo was a final project for CSCI380: User Interface Development in Fall 2020.
